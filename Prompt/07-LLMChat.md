TUI내 챗봇기능 상세설명
## 
기존 TUI의 초기메뉴에 Chat 기능이 추가되며...TUI의 터미널 환경에서 AI를  이용한 자연스러운 채팅을 진행할수 있습니다. 
LLM은 자동다운로드가 아닌 이미 다운로드된 위치를 선택할수 있어야하며
어플리케이션 프로파일 기본 LLM 디렉토리가 기본경로로 채택합니다. 모델이 없을때 ChatBot은 해당 모델을 다운로드 받아야한다고
경로로 안내합니다.

주의 사항:
- 이 바이너리는 멀티OS AOT를 지원하며 특정 OS종속(예>윈도우만작동가능) 적인 빌더를 채택하지 않습니다. 윈도우/맥/리눅스 환경에 빌더될수 있는 모듈을 채택합니다.
