다음은 예시이며 윈도우와 리눅스가 일반적으로 사용하는 개인사용자(id) 디렉토리를 활용합니다.

### CLI
CLI명령은 예시이며 CLI명령이 추가되는경우 아규먼트포함 더 좋은 네이밍이 있으면 제안해 조정합니다.
- codescan search 시, project id 아규먼트 추가해 해당 프로젝내 검색을 추가합니다.
- codescan project-addinfo 명령을 추가합니다. 아규먼트는 project-id 이며 해당프로젝트에 추가설명을 할수 있습니다. 단하나의 설명만 추가가능 
- codescan project 는 특정프로젝트의 수집된 정보를 모두 보여줍니다.
  - addinfo가 있는지 없는지 보여줍니다.
  - 없는경우 , 이 정보를 LLM이 분석해 보강할수 있도록 프롬프트로 유도합니다.
