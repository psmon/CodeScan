## 개선지침

### CLI
CLI명령은 예시이며 CLI명령이 추가되는경우 아규먼트포함 더 좋은 네이밍이 있으면 제안해 조정합니다.
- 등록한 project를 delete하는 명령이 없습니다. 추가해주세요
- 등록한 project의 구성요소 내용 update(항목별부분업데이트) 하는 명령이 없습니다. 하위요소를 체크해 추가해주세요
- project id 로 정보를 조회할때 상세내용이 너무많은경우 아주 길어집니다. , 해당기능은 썸머리로 제공하고 , detail 아규먼트가 붙었을때 해당항목이 보여질수 있도록 cli명령어를 개선해주세요


### TUI
- CLI의 기능업데이트에 맞게 TUI도 해당기능을 동일하게 이용할수 있게 업데이트

- 빌드오류가 없는지 체크합니다. (이 환경은 윈도우환경의 파워쉘 위에서 구동되고 있습니다.)
- Scan 테스트는 이 경로자체를 이용합니다. (CodeScan 프로젝트 - 자기자신을 codescan이 설명할수 있는 셀프테스트구조)

- 테스트가 완료되면 이번 지침 수행으로 추가되거나 변경된 기능 , AGENT.md 에 설명합니다.
