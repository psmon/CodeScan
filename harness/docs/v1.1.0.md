# v1.1.0 — 초기 전문가 3명 투입

## 변경 요약

CodeScan 프로젝트 성격(정규식 기반 다국어 파서 + Native AOT + xUnit)에 맞춘 초기 전문가 3명을 정원에 심었다.

## 추가된 요소

### 에이전트 (agents/)

| 이름 | 역할 |
|------|------|
| regex-safety-guard | `[GeneratedRegex]` AOT 안전성, 백트래킹 위험, 7개 언어 패턴 검수 |
| aot-compatibility-scout | Native AOT 호환성 — 리플렉션/동적 코드/IL 경고 탐지 |
| test-sentinel | SourceAnalyzer/CommentExtractor 언어 × 케이스 매트릭스 점검 |

### 지식 (knowledge/)

- `regex-patterns.md` — 7개 언어 정규식 패턴 가이드, AOT 위반/백트래킹 패턴 목록
- `aot-rules.md` — Native AOT 배포 규칙, 금지/주의 API, IL 경고 코드 표

### 엔진 (engine/)

- `test-audit.md` — 세 에이전트가 협업하여 언어 × 케이스 매트릭스를 점검하는 워크플로우

## config 변경
